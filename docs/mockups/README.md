# Pocket Control Room Mobile Mockups

These phone-sized studies explore a skeuomorphic mobile direction built from Little Control Room's existing pixel operator, signal colors, and control-station hardware.

- `mobile-pocket-switchboard.png`: project and attention dashboard
- `mobile-pocket-project.png`: project overview centered on the active engineer
- `mobile-pocket-session.png`: live transcript and approval interlock

The source is `internal/tui/mobile_console_mockups.go`. Regenerate the complete mockup catalog with:

```sh
make mockups
```
